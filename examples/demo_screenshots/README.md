# Atlas Mercator — Demo Screenshots

Drop PNG/JPG/GIF captures of the Gradio demo here.  Filename convention:

```
NN-<tab>.png
```

Where `NN` is the order in the README carousel and `<tab>` is the
Gradio tab name.

Recommended captures (run `atlas-demo` to generate them):

1. `01-orchestrator.png` — Atlas Orchestrator tab with a workflow running.
2. `02-listing.png` — Listing Optimizer tab with a 5-segment listing.
3. `03-support-rag.png` — Customer Support tab showing RAG citations.
4. `04-marketing.png` — Marketing Copilot tab showing 3 A/B variants.
5. `05-intel.png` — Intel Scout tab showing threat score + differentiators.
6. `06-kb.png` — Knowledge Base tab with semantic search results.

Each image should be ~1280×800 for retina-friendliness; keep file size
under 1 MB (use `pngquant` or similar).
